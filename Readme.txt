To install the Admin Dashboard

# run front End Dashboard
navigate to /dashboard and run : npm install 
it will install the react dependencies

then run : npm start 
to run the react server


To install node server 

# run Backend
navigate to /server run : npm install 
to install node dependencies

then run : node app.js
 to run the server


 # run Socket server 
 navigate to /server run : node server.js
